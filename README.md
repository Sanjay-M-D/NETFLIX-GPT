# Steps for Deployment in Firebase

- Install Firebase CLI ---> 'npm install -g firebase-tools'
- Firebase Login ---> 'firebase login'
- Initilize Firebase ---> 'firebase init' , then select Hosting
- Deploy command ---> 'firebase deploy'
